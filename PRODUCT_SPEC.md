# FanSpot MVP Product Spec

## Product idea

FanSpot is a creator-focused social platform where fans can follow creators, discover new accounts, view posts, receive updates, and eventually support creators through approved subscription/payment workflows.

## MVP user types

### Fan

- Browse the homepage
- Create an account
- Follow creators
- View a feed
- Message placeholder
- Manage profile/settings

### Creator

- Apply to become a creator
- Manage profile
- View audience stats
- Draft posts
- Review earnings placeholder

### Admin

- Review reports
- Review creator applications
- Review content flags
- Review payout queue
- View platform metrics

## MVP design direction

- Dark-first
- Premium black/blue/white palette
- Social feed inspired layout
- Left navigation on desktop
- Bottom/mobile navigation later
- Right rail recommendations on desktop
- Rounded cards, blue CTAs, soft borders
