# Environment Variables

Required for production:

- DATABASE_URL
- NEXTAUTH_URL
- NEXTAUTH_SECRET
- AWS_REGION
- AWS_ACCESS_KEY_ID
- AWS_SECRET_ACCESS_KEY
- AWS_S3_BUCKET
- AWS_CLOUDFRONT_URL
- SEGPAY_WEBHOOK_SECRET or processor equivalent

Recommended:

- APP_ENV
- ADMIN_EMAILS
- SUPPORT_EMAIL
- RATE_LIMIT_WINDOW_SECONDS
- RATE_LIMIT_MAX_REQUESTS
- LOG_LEVEL
