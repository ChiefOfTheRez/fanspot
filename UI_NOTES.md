# UI Notes

## Visual direction

- Dark-first premium interface
- 60/30/10 color rule: black base, blue accents, white text/highlights
- Rounded cards
- Blue CTAs
- Instagram-style feed structure
- Desktop sidebar and right recommendations rail
- Mobile bottom nav shell

## Important MVP screens

- Homepage should sell trust and creator quality
- Feed should feel active even before features are connected
- Discovery should highlight founding creators
- Creator profiles should clearly show verified/founding badges
- Studio should help creators know what to do next
- Admin should make safety and operations obvious

## Later polish

- Real light mode
- Sidebar expand/collapse animation
- Better skeleton loaders
- Toast notifications
- Modal system
- Real image/video previews
- Creator profile themes
